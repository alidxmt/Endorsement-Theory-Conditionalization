Endorsement Theory

Ali Zolfagharian - 'What If' Research Group - Konstanz University - 2019

Endorsement theory, ET, is a formal epistemological theory which represents an epistemic state of an agent by an acceptance base function (ABF), an endorsement critical point (ECP), and an endorsement function (EF). ABF, ECP, and EF respectively depend on agent’s evidence, context and reasons.

The static ET ends by representing qualitative and quantitative doxastic attitudes, namely; belief, disbelief, suspense and ignorance. For the dynamic part, I explain how an agent can update her acceptance base function, endorsement threshold and her way of constructing the endorsement function.

The last part is about conditionals and reason. Conditional degree of endorsement of proposition, depends on the given condition, a condition could be change in the acceptance base. Reason on the other side is about comparing how a condition can change the degree of endorsement of the proposition and its complement.

Preliminary definitions and notation
Set of possibilities and propositions

Let W be the set of all possibilities and n be the number of all possibilities. Then Pi ⊆ W is a proposition, a set of possibilities; such that the subscript i is a decimal number which its binary form indicates which possibilities are in Pi. The notation, using subscript i, improves the readability of the text and it is easy to use this notation for a programming language like bitwise operator. Also, it reflects the definition of a proposition, a function from the set of possibilities to {0,1}.

  • (1) W = { w 1 , w2 ,…, wn }
  • (2) Pi ⊆ W

Let me explain with an example: Assume W={ w1 , w2 , w3 , w4 }, then the proposition W is P15 which is P(1111), the number 1111 tells that all possibilities w1,, w2,, w3,, w4 belongs to P15. Similarly, P5 ≡ P(0101) which tells that P5 contains w1,, w3. And P0 means the empty set.

Frames and ground

The set of frames F is the set of all algebra over W. Fi is a subset of F, that all its member contains Pi. In other word Fi is the set of all algebra over W which have Pi as its member. An algebra could be thought as an inner context or a framework that has enough position to receive evidences. There might be cases that the inner context needs to be updated. Let F be { Bi | Bi is an algebra and a subset of 2W } then

  • (3) Fi | ( Fi ⊆ F ) ∧ ( Pi ∈ Fi )

The set of grounds G are the set of sets S such that each S, is a set of propositions and if it had more than one element then it has three properties: I) The intersection members of any subset of S does not belong to S. II) The intersection of all members of S is not the empty set. Let the intersection of members of S, be called a core. The set of all cores is the set of all propositions as well. III. There is no subset of S with the same core.

Each member of G for the set of all proposition 2W, could be called a prime ground for the set of all grounds . Gi is a subset of G that contains all S in G that Pi is the core or a superset of the core of S.

  • (4) Gi = { Si | Si ⊆ 2W, ∩S ⊆ Pi }
Acceptance Base Fucntion
Evidence

A peace of evidence changes epistemically agent’s degree of acceptance. A piece of evidence about Pi changes the acceptance degree of Pi. Notice that accepting a proposition at some degree does not mean that the agent necessarily uses that proposition for reasoning.

Evidence

An acceptance base function δ is a function from 2W to [0,1] or δ: 2W → [0,1] such that

  • (5) δ ( ∅ ) = 0
  • (6) δ ( W ) = 1

This function assigns 0 if the agent does not have evidence in favor of the proposition and assign a number from (0,1] if she has evidence. For combining n pieces of evidence which are in favor of a proposition like Pi and each piece of evidence supports Pi with a certain degree of acceptance E j , one can use the following rule of combination:

  • (7) δ ( P i ) = ∑ I⊆{E1,E2,...En} (-1) |I|+1 ∏ E j∊ I E j
For two pieces of evidence E 1=0.4 and E 1=0.8, the result is δ(P i = E1 + E2 - E1 × E2 = E1 + E2 (1 - E1) = 0.4 + 0.48 =0.88. It means each piece of evidence increase the percentage of the remaining degree of acceptance.

If a proposition does not have evidential support, then its degree of base acceptance is 0. The acceptance base function reflects evidence or data before the data processing.

Endorsement Fucntion
Endorsement Threshold

An endorsement threshold is a number Cr in [0,1], that the agent endorses a proposition if its acceptance base is more than Cr. An agent might have different threshold in different context.

Endorsement Base

The set H of all propositions that their degree of base acceptance is equal to or more than the threshold is the endorsement base. The agent uses elements of H as the foundation of her reasoning and endorsement:

  • (8) H = { P i | δ ( P i ) ≥ Cr }
Inferable Bases and cores

A set Hi is an inferable base if it is consistent and contain all propositions in H that they are compatible with Hi. In other words, an inferable base is one of the largest consistent subsets of H. The subscription of Hi shows that the intersection of all propositions in Hi is Pi = ⋂Pk ∈ HiPk.

  • (9) H i ⊆ H
  • (10) P i ≠ ∅
  • (11) ∀ Pk ∈ H (Pi ∩ Pk≠ ∅ → P k∈ Hi)

For example, if |W|=4 and H = { P2, P3, P10, P12 }, then H = { P2, P3, P10, P12 } is not an inferable base because its core is P0; and also the set { P2 , P10 } is not an inferable base because it is not the largest consistent subset of H and by adding P3 still its core is not the empty set. The inferable bases of H are: H2={ P2 ,P3 ,P 10 } and H8={ P 10, P 12 } and their corresponding core are respectively P2 and P8.

Conditionally Endorsed propositions

A proposition Pj is endorsed in an inferable base Hi if and only if Pi ⊆ Pj.

  • (12) End ( Pi | Hi ) ≡ Pi ⊆ Pj

End ( Pi | Hi ) means if the agent derives propositions based on Hi then Pj is endorsed. End is the qualitative or binary endorsement. A proposition in a given inferable base is endorsed or is not endorsed. If it is endorsed, then its complement is not endorsed. If it is not endorsed, then its complement is endorsed or is not endorsed.

Conditionally Evaluable propositions

Let ¬Pj be W - Pj. A proposition Pj is evaluable in an inferable base Hi if and only if Pi ⊆ Pj or Pi ⊆ ¬Pj. As the core of an inferable base is not the empty set, a proposition and its complement cannot be endorsed in an inferable base at the same time.

  • (13) Eva( Pj | Hi ) ≡ Pi ⊆ Pj or Pi ⊆ ¬Pj

Eva( Pj | Hi ) means then Pj is decidable in Hi. If a proposition is evaluable in the inferable base Hi, then the proposition or its complement is endorsed in Hi. Therefore:

  • (14) Eva( Pj | Hi ) ≡ Eva( ¬Pj | Hi )
Evaluable propositions

A proposition Pj given H; the endorsement base, is evaluable, if and only if it is conditionally evaluable in all inferable bases of H.

  • (15) Eva( Pj ) ≡ Eva( Pj | H ) ≡ ∀ Hi Eva( Pj | Hi )

The set of all evaluable propositions; V, is an algebra. Its proof is simple. For every evaluable proposition its complement is evaluable (14). And the union (and intersection) of two evaluable propositions are evaluable (13). So, V is an algebra.

  • (16) V ≡ { Pj | Eva( Pj ) }
  • (17) V ∈ F
Endorsed propositions

A proposition Pj given H; the endorsement base, is endorsed, if and only if it is evaluable and it is conditionally endorsed in at least one inferable base of H.

  • (18) End(Pj ) ≡ End(Pj | H) ≡ Eva(Pj ) ∧ ∃ Hi End (Pj | Hi )
Endorsement function

Endorsement Function gives a degree of endoresement to a proposition and it represents the quantitative notion of endorsement. The degree of endorsement of a proposition is the maximum degree of support that it could be received from an inferable base. In other words, the degree of endorsement, should be assigned based on the best consistent argument that an agent with H as her endorsement base could give.

Every proposition has a set of possible grounds Gi. Let GiS, be the set of all possible grounds of Pi such that every proposition in the elements of the sets in G, is an element of S. Then, GiS is the set of grounds for Pi that could are provioded by S. Now, GiH means all grounds that could be provided for Pi based on the Endorsement Base. In the same way, GiHj is the set of all grounds for Pi which an inferable base could provide. If we consider every ground S, which is a set of propositions, in Hj as an evidential support for Pi provided by Hj, then the degree of endorsement, Endo, that it gives to Pi is the following:

  • (19) Endo( Pi | S ) = min {δ ( Pj ) | Pj ∈ S }

The reason that we need a minimum function, is that the ground is as supportive as its weakest proposition. In our daily life, when we hear an argument that one of its assumptions is not acceptable or is weakly acceptable, we think the whole argument is weak as well. Notice that the minimum for all grounds in GiH is more than the endorsement threshold, and it means if the ground is provided by H then it is gives aminimum more than the threshold.

For calculating the degree of evidential support from a set of grounds GiHj, one can aggregate all supports from various grounds by using the following formula:

  • (20) Endo( Pi | Hj ) = ∑ I⊆GiHj (-1) |I|+1 ∏ S j∊ I Endo( Sj )

The above formula works like the evidential support formula that we used for the acceptance base function. Now, we are able to make the endorsement function Endo( Pi ) or Endo( Pi | H ) : (maybe Aggr is a better fomulation again)

  • (21) Endo( Pi ) = max { Endo(Pi | Hj ) | for all Hj }
Doxastic attitudes
Belief

A proposition is believed if and only if it is endorsed and its negation is not endorsed:

  • (22) Belo( Pi ) = End( Pi ) ∧ ¬End( ¬Pi )
Disbelief

A proposition is disbelieved if and only if its negation is believed:

  • (23) Diso( Pi ) = Belo( ¬Pi )
Suspension

A proposition is suspended if and only if the proposition and its negation are both endorsed:

  • (24) Suso( Pi ) = End( Pi ) ∧ End( ¬Pi )
Ignorance

A proposition is unknown if and only if it is endorsed and its negation is not endorsed:

  • (25) Igno( Pi ) = ¬End( Pi ) ∧ ¬End( ¬Pi )
Update
Acceptance Base Function Update

As per changes in the evidential support of propositions, one can update the acceptance base function by considering its restriction (5) and (6), and by using the rule of combination. This update, is natural, as we do in daily life. The revision in the acceptance function, necessitates the re-calculation of endorsement function as well.

Endorsement Critical Point Update

Updating the critical point, means assigning new number between 0 and 1. After the update, the acceptance function does not need to be re-calculated, but the endorsement function needs re-calculation. The motivations for the update of the critical point might be based on some practical or epistemological reasons.

Update in the Construction of the Endorsement function

The logic of deduction, could be updated as well, if an agent does not accept the classic logic. Also, an agent could change the norms for making inferable sets. A change in logic, necessitates re-calculation of endorsement function.

Conditionals
By assuming endorsed/not endorsed

Assuming endorsed. If an agent has endorsed a propostion, by assuming endorsed, her endorsement function, will not change. If the proposition is not endorsed, then she can endorse it by assiging Cr as its degree of acceptance. Then she should update her endorsement function again. In this case, one can call it 'conditioning by assuming endorsed'.

  • (26) Endo( . | Endo( Pi ) ) = Endo( . | δ( Pi ) = Cr )

One can say that another option for the agent might be decreasing her critical point if it helps endorsing the proposition. In this case there is no act of assumption, but the agent only change what still epistemically is acceptable. What is important here, is the first case. Update of the threshold might have many reasons and endorsing a proposition is one of many possible reasons.

Then she could avoid all arguments that make the propostion endorsed. Let Gi be the set of all ground of Pi :

By Believing

Assuming that Pi is believed is equal to assume that Pi is endorsed (26) and ¬Pi is not endorsed (27).

By Disbelieving

Assuming that Pi is disbelieved is equal to assume that Pi is believed.

By Assuming Unknown

Assuming that Pi is unknown is equal to assume that Pi and ¬Pi are not both endorsed (27).

By Suspending

Assuming that Pi is suspended is equal to assume that Pi and ¬Pi are both endorsed (26).

Conditional Endorsement

Endo( Pi | Pj ) is maximum degree of endorsement of Pj in an inferable base where Pj is endorsed as well. If there is no inferable base that Pj is endorsed, then Endo( Pi | Pj ) is undefined.

  • (28) Endo( Pi | Pj ) = max {Endo( Pi | Hk ) | Hk endorses Pj }

The difference between this conditional and above conditionals, is that they are conditionals based on assuming, and in this case it is conditional based on reasoning and evidential support that agent has for a proposition on the given condition.

Reasons
various kinds of reasons

Reason is about agent's structure of endorsing various propositions. A proposition Pj is an epistemic constructor for Pi, if Pj is an element of the ground which gives the Pi a degree of endorsement, or if it contributes directly in an argument for Pi. Let Pj ⊳ Pi be Pj constructs Pi, then

  • (29) Pj ⊳ Pi ≡ ∃ S ∈ GiH (Pj∈ S)

One can observe that there are four possible relationships between two propistions: A. Pj ⊳ Pi, B. Pj ⊳ ¬Pi, C. ¬Pj ⊳ Pi, D. ¬Pj ⊳ ¬Pi. As they also could happens at the same time, there are 16 possible relationships between two propositions.

If a proposition Pi could be believed, disbelieved, suspended or unkown. If Pi is believed, then Pj is a reason for believing Pi, if and only if, Pj ⊳ Pi. If Pi is disbelieved then Pj is a reason for disbelieving Pi if and only if Pj is a reason for believing ¬Pi. If Pi is suspended, then Pj is a reason for suspense of Pi, if and only if, Pj ⊳ Pi and Pj ⊳ ¬Pi.

If Bel ( Pi ) then Pj is a sufficient reason of believing, if and only if ( ¬Pj ⊳ Pi ). If Dis ( Pi ) then Pj is a sufficient reason of dibelieving, if and only if it is sufficient reason for believing ¬Pi. If Sus ( Pi ) then Pj is a sufficient reason of suspending, if and only if ( ¬Pj ⊳ Pi ) and ( ¬Pj ⊳ ¬Pi ).
